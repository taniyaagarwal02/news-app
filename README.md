# 📰 News App (Android)

A modern Android news application built using Kotlin and Jetpack Compose that fetches real-time news using REST APIs. The app demonstrates clean architecture, scalable architecture, and responsive UI with smooth user experience.

---

## 🚀 Features

- 📰 Fetch latest news from API
- ⚡ Real-time data using Retrofit + Coroutines
- 🧭 Navigation between screens
- ⏳ Loading states and error handling
- 🎨 Modern UI with Jetpack Compose & Material Design
- 📱 Responsive and user-friendly interface
- 🧩 MVVM architecture for clean code structure

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Retrofit
- Coroutines
- Android SDK
- Material Design

---

## 📂 Project Structure

- **ui/** — Contains screens and Jetpack Compose components  
- **data/** — Includes models, repository, and API integration  
- **viewmodel/** — Handles UI state and business logic  
- **utils/** — Utility and helper functions  

---

## ⚙️ How to Run

1. Clone the repository

git clone https://github.com/taniyaagarwal02/news-app.git

2. Open project in Android Studio
3. Sync Gradle
4. Run on emulator or physical device
